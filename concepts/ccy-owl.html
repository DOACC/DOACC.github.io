CCY in OWL

Ontology for a cryptocurrency, based on Melvin Carvalho’s “Ontology for Crypto Currencies” with a few adjustments and additions.

Melvin Carvalho’s parent ontology was developed before the recent explosion in numbers of altcoins and so the model understandably reflects some Bitcoin-specific aspects. This version makes a couple of judicious changes to allow for a general altcoin model to be used.

Concepts in this ontology are mainly focused on modelling the blockchain, headers, transactions, etc. The model has been extended to include relations for expressing a number of distinguishing program variables and parameter bindings.
