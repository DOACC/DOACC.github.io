Privacy Policy
Privacy inherited from Github

This web site is a collection of static pages of HTML hosted on “Github pages” and so inherits Github Pages privacy policy, see https://help.github.com/articles/github-privacy-policy/ for details.

No additional data is collected beyond the standard request log maintained by Apache’s request logging which is used for occasional trouble-shooting.

Javascript libraries used in the examples are resourced either locally to the server or from standard ssl-providing content delivery networks.
